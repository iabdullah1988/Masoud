# هيئة الزكاة والضريبة والجمارك (ZATCA)
# إطار حوكمة الامتثال الضريبي

## 1. سياسات الامتثال الضريبي
- سياسة داخلية لإدارة مخاطر الضرائب
- إجراءات الالتزام بضوابط الفوترة الإلكترونية
- مسؤول امتثال ضريبي معين

## 2. الفوترة الإلكترونية (فاتورة)
- تكامل مع منصة فاتورة
- إصدار فواتير إلكترونية متوافقة
- حفظ سجلات الفواتير حسب المتطلبات

## 3. حفظ السجلات (SAF-T)
- الاحتفاظ بالسجلات المحاسبية إلكترونياً
- تنسيق متوافق لمتطلبات ZATCA
- إتاحة السجلات للتدقيق عند الطلب

## 4. الإقرارات الضريبية
- تقديم إقرارات دقيقة وفي الوقت المحدد
- مراجعة داخلية قبل التقديم
- احتساب وتحصيل الضريبة بشكل صحيح

## 5. التدقيق الداخلي
- مراجعة دورية للامتثال الضريبي
- تقارير ربع سنوية للإدارة
- خطة تصحيحية للملاحظات
