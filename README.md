# مسعود - Masoud AI
### الحوكمة والامتثال الذكي للأنظمة السعودية

## عن المشروع
منصة ذكاء اصطناعي لمساعدة الشركات على الامتثال للأنظمة السعودية من خلال:
- تحليل مستندات الشركة
- المقارنة مع اللوائح الرسمية
- اكتشاف الفجوات وتقديم التوصيات

## الجهات الرقابية المشمولة
1. هيئة السوق المالية (CMA)
2. البنك المركزي السعودي (SAMA)
3. الهيئة الوطنية للأمن السيبراني (NCA)
4. الهيئة السعودية للبيانات والذكاء الاصطناعي (SDAIA)
5. هيئة الحكومة الرقمية (DGA)
6. هيئة الزكاة والضريبة والجمارك (ZATCA)
7. وزارة التجارة

## التقنيات المستخدمة
- Python + FastAPI
- React
- Oracle Cloud Free Tier
- نماذج Llama/Gemma مفتوحة المصدر
